本書について

  • 本書は，Ruby on Rails Tutorial の日本語訳です．
    原著と同様に，オンライン版は無料で公開しています．
    本書が皆さんの Rails 学習の一助になれば幸いです．
    　　　　　　　　　Rails Tutorial 翻訳グループ一同

  • 最近の進捗状況:

    　　Rails 4.0 にも対応しました！
    　　右側にあるサイドバーから Rails 3.2/4.0 の切り替えができるようになっています！
    　　また，iPhone/Android などのモバイル端末にも対応しました :D
    　　今後も原著が更新され次第，随時こちらの日本語訳も更新していく予定です．

  • ご協力のお願い:

    　　「○○に対応して欲しい！」などのご要望がありましたら
    　　@yasulab までお気軽にご連絡ください．
